3.2 核心教训（本周反复验证）

1. 赚钱效应是检验情绪的唯一标准（教训#5，本周3次验证）

  • 3/31：情绪71%→18%脉冲，赚钱效应差=诱多 ✓
  • 4/1：情绪93%→81%脉冲，赚钱效应-0.18%=诱多 ✓
  • 4/2：情绪81%→19%崩塌，赚钱效应1.09%=虚假高潮 ✓

2. 脉冲修复>50pp = 诱多（教训#6延伸）

  • 3/31：34%→71%（+37pp）= 诱多 ✓
  • 4/1：18%→93%（+75pp）= 诱多 ✓
  • 本周两次脉冲修复全部是主力出货，追进去的人全亏

3. 龙头独秀+跟风杀跌 = 退潮信号（教训#6，本周2次验证）

  • 3/31：神剑4板但航天工程-0.68%、航天发展-0.77% → 航天退潮 ✓
  • 4/2：津药5板但凯莱英-2.13亿、睿智-7276万 → 医药内部大迁徙 ✓

4. rules不满足 = 空仓，没有例外（本周血泪教训）

  • 4/2：冰点+7项仅1项满足 → 窗口1追涨睿智医药 → 亏损-2952元（-7.58%）
  • 这是本周唯一的亏损来源，且100%由违规操作造成

5. 双冰环境绝对空仓

  • 连续两天情绪<20% = 市场极度恐慌
  • 赚钱效应归零、跌停>涨停、主力-232亿
  • 双冰后第三天大概率继续弱，不赌反弹

3.3 待固化的认知

1. 光通信/光纤是新方向：有产业逻辑（AI智算中心+光纤涨价+英伟达硅光子），走趋势非连板，主力资金大量逆势流入（通宇+12亿、天孚+9.6亿）。但当前rules无双冰+趋势方向确认标准，待短线生态恢复后评估纳入

2. 假设空仓测试（教训#30）：如果空仓不会买→但方向未退潮→收紧止损观察（待验证中）

3. 涨停板+主力出逃的双解读：可能是出货也可能是接力资金换手，关键看次日竞价验证（4/3新能泰山案例）

6.2 周一（4/7）重点观察

1. 双冰后修复方向：强修复（40%+）还是弱修复（20-35%）还是继续冰（<20%）？

2. 光通信能否持续：新能泰山4板？板块涨停≥8只+有梯队+有弹性（复合条件）？主力是否继续流入？

3. 津药药业7板还是断板：断板=医药彻底结束，封板=独苗续命

4. 周末消息面：有无AI/算力/光通信利好？有无重大利空？

5. 赚钱效应：今天涨停38只周一能给多少溢价？（昨涨停收益关键指标）

6. 主力资金：能否止血（本周连续流出-232亿）？
