第三章交易体系深度解构

「不轻易出手，出手就是精品，出手就是龙头，这些龙头都是市场公认的龙头，而不是像我们买了个票就意淫会不会成为龙头。」
①龙头战法（不变的核心）▼

只做市场公认的绝对龙头。不参与杂毛题材，不追逐「可能成为龙头」的意淫型标的。首板操作通常以套利思维参与板块大爆发。

②趋势锁仓（区别于纯短线游资的关键）▼

确定性高的主升浪行情中敢于重仓锁仓，持仓周期可达数周乃至更长（如赛力斯跨年度持有）。跌停不割肉：短期调整甚至跌停也倾向于等待板块爆发。

③基本面深度结合▼

偏好有强逻辑支撑的行业龙头。选股时基本面是前置条件，而非事后找理由。代表作：赛力斯=新能源车转型，剑桥科技=CPO光模块。

④高位容量股二波形态▼

擅长操作高位仍有空间、能形成二波上涨的容量股。不是追高，而是在确认二波形态走出来后跟随介入。偏好成交额大、流动性好的大盘龙头。

⑤板块共振择时▼

倾向于在指数上升期或题材共振期重仓介入。善于捕捉「情绪冰点+个股辨识度」形成的买点。

第九章研究结论

毛老板的独特定位：格局之王 / 造妖转型趋势的典范 / 出手即精品

① 「从造妖到趋势」转型最彻底的游资：早期金田路短线妖股，后主动转向基本面趋势投资
② 锁仓能力最强：即便跌停也坚持不割，等板块爆发——顶级游资中极为罕见
③ 出手即精品：不轻易出手，一旦出手必押注市场公认龙头

对弈沐的实战价值

→ 龙虎榜出现「国泰君安北京光华路」净买入 → 大概率是中长期格局型操作
→ 毛老板重仓某主线龙头 → 可作为中线趋势行情的参考信号
→ 毛老板跌停仍坚守 → 说明对该股产业逻辑有深刻理解
→ 毛老板「出手即精品」理念 → 可强化弈沐「无信号不动」的铁律
